# 説明
ROS2 ノード
talkerで0.5秒ごとに１づつカウントアップされる数値を
listenerに送信し端末に出力するノード

#実行方法
次の操作はROS2ワークスペースのトップディレクトリで行う
```
$ colcon build
$ . install/setup.bash
$ . install/local_setup.bash
$ ros2 launch mypkg talk_listen.launch.py
```

#必要環境
* ROS2
* Python 3

#テスト済み動作環境
* ROS2 Humble
* Ubuntu 22.04
* Python 3.10.6
